# Preproposal

## What idea(s) do you have for your final project?

My idea is to use Arcade or PyGames to create a sci-fi variant of the Uncharted games, one of my favorite franchises of all time. 
The project would inclulde multiple puzzles that require the user to interact with objects in the environment to collect information that they need to solve the puzzle. I would love to do like a fancy combination lock that requires the user to acquire decryption keys from different parts of a location to figure out what the combination is, or something like that. 

I wanted to make it sci-fi because it would encourage me to make it more creative from a storyboarding perspective, and since it isn't necessarily grounded by regular physics anymore, I could be more flexible with how I want the story and chracters to work.
It would also include platforming and combat sections on top of the puzzle sections. I also want to have elements that the game keeps track of, such as mechanisms like damage taken, a hunger mechanic, rage mechanic, etc. that way it's a little more dynamic of a gameplay and keeps the user interested. 
I also would want to use objects like journals, which are in the Uncharted game, where once a user interacts with specific, predetermined objetcs, data is added to their journal that they can reference at any time that will help them sovle the puzzles, interact with the environment, or reveal enemy weaknesses. I think it might also be cool to have a mechanic where the character's healing is predicated on damaging the enemy, encouraging an active form of gameplay.

This is really tentative, but I also would like to recreate an iconic scene from the Uncharted franchise where Nathan Drake (the titular character) is fighting a helicopter on a railway car. I'd obviously make this sci-fi, but I think it would be really fun to recreate one of favorite (yet also incredibly darn frustrating) parts of the game. 

This project is a great way to learn game design while focusing on core gameplay loops using the stuff we learned in class like for and while loops. Furthermore, I'll do my best to implement something like Monte Carlo-reminiscent code to perhaps make one of the puzzles or something.
